# Contributing to AgroGuardAI

Thank you for your interest in contributing to AgroGuardAI.
Every contribution directly helps farmers across Africa.

---

## Code of Conduct

By contributing you agree to be respectful, inclusive and 
constructive in all interactions.

---

## How to Contribute

### 1. Fork the Repository
Click the Fork button on GitHub to create your own copy.

### 2. Clone Your Fork
git clone https://github.com/YOUR_USERNAME/AgroguardAI.git
cd AgroguardAI
### 3. Create a Branch
git checkout -b feature/your-feature-name
### 4. Make Your Changes
Write clean, well-commented code.
Test your changes before submitting.

### 5. Commit Your Changes
git commit -m "Add: brief description of your change"
### 6. Push and Submit Pull Request
git push origin feature/your-feature-name
Then open a Pull Request on GitHub.
