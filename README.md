# 🌿 AgroGuardAI

> **An open source AI crop doctor for African smallholder farmers.**

A farmer points their phone at a sick plant, speaks in **Hausa, Igbo, Yoruba or Swahili**, and receives a spoken diagnosis and treatment plan back. No reading required. No agronomist needed. No internet required (EdgeAI — coming soon).

## 🚨 The Problem

- Nigeria has **39 million smallholder farmers**
- **40% of crops** are lost annually to pests and diseases
- Expert advice is locked in English, in cities, far from farmers who need it
- Existing tools require literacy, internet, and smartphones most farmers don't have

---

## 💡 The Solution

AgroGuardAI is a **dual-force precision agriculture system** built in two layers:

### Layer 1 — AI Crop Doctor (Software) ✅ In Development
- Voice-first interaction — farmers speak, the app speaks back
- Supports **Hausa, Igbo, Yoruba and Swahili**
- Image-based disease diagnosis powered by Gemini Multimodal API
- Designed for low-cost Android devices
- **90% diagnosis accuracy** validated by KATDICT (Katsina State Directorate of ICT)

### Layer 2 — Agrobotics (Hardware) 🔜 Roadmap
- Autonomous drones and ground rovers
- EdgeAI on-device inference using NVIDIA Jetson Orin Nano
- Real-time pest outbreak detection across large farmlands
- Works completely offline in deep rural areas

## 🚀 Getting Started

git clone https://github.com/mohammerdzakari/AgroguardAI.git
cd AgroguardAI

# Install dependencies
npm install

# Add your Gemini API key to .env
GEMINI_API_KEY=your_key_here

# Run the app
npm run dev
